# 💧 喝水提醒应用

一个纯原生安卓喝水提醒应用，支持自定义铃声、个性化提示、勿扰时段等功能，可在后台常驻运行。

## ✨ 功能特性

### 核心功能
- ⏰ **自定义提醒间隔** — 支持 1-180 分钟灵活设置
- 🔔 **铃声选择** — 从系统铃声库中自由选择提醒铃声，支持预览播放
- 📳 **震动提醒** — 三段式震动模式
- 📱 **通知栏提醒** — 完整的通知栏消息，点击可返回应用
- 🔄 **自动循环** — 提醒过后自动进入下一个计时周期
- 💾 **数据持久化** — 所有设置自动保存

### 高级设置
- 🔇 **勿扰时段** — 支持设置夜间和午休两个免打扰时段，可自定义起止时间，开启后该时段内不会发送提醒
- 💬 **个性化提示** — 自定义通知标题和内容文字，如"该喝水啦！"或"放松一下眼睛吧~"
- 🎵 **铃声预览** — 高级设置中可直接点击预览按钮试听当前选中的铃声
- ❓ **权限帮助** — 内置多品牌手机的权限设置指引

### 后台保障
- 🔄 **前台服务** — 使用 Foreground Service 确保后台稳定运行
- ⏱️ **AlarmManager 闹钟** — 系统级精确定时，熄屏/后台依然准时提醒
- 🔃 **开机自启** — 设备重启后通过 BootReceiver 自动恢复提醒
- ⏸️ **真正暂停** — 暂停时同时停止后台服务和取消系统闹钟，杜绝误提醒

## 🚀 编译方式

### 方法一：GitHub Actions 自动编译（推荐）

1. 将代码推送到 GitHub 仓库
2. GitHub Actions 会自动触发构建
3. 在仓库的 **Actions** 页面 → 选择 **Build Android APK** → 点击最新运行记录
4. 在页面底部的 **Artifacts** 区域下载 `app-debug.apk`

### 方法二：本地编译

```bash
# 安装 Java 17（如果未安装）
sudo apt install openjdk-17-jdk  # Ubuntu/Debian
brew install openjdk@17          # macOS

# 进入安卓目录
cd android

# 编译 debug 版本
./gradlew assembleDebug

# 编译 release 版本
./gradlew assembleRelease
```

## 📱 APK 位置

```
android/app/build/outputs/apk/debug/app-debug.apk
android/app/build/outputs/apk/release/app-release.apk
```

## 📖 使用说明

### 快速开始

1. 打开应用，默认为 20 分钟提醒间隔
2. 点击 **开始** 按钮启动提醒计时
3. 到达设定时间后会自动弹出通知、播放铃声并震动
4. 点击 **喝完了** 记录喝水时间并重置倒计时
5. 点击 **暂停** 停止计时和后台提醒

### 高级设置

点击主界面右上角的 **齿轮图标** 进入高级设置页面：

- **提醒间隔**：通过 +/- 按钮或直接输入调整分钟数（1-180 分钟）
- **提醒铃声**：点击"选择铃声"从系统铃声库中挑选；点击 ▶ 按钮预览当前铃声
- **个性化提示**：自定义通知栏显示的标题和内容文字
- **勿扰时段**：开启后可设置夜间和午休两个免打扰时段，点击时间按钮即可调整
- **权限帮助**：查看各品牌手机的电池优化设置步骤

### 测试提醒

点击主界面的 **测试** 按钮，立即触发一次提醒，方便验证铃声和文字设置是否生效。

## 📋 权限说明

| 权限 | 用途 |
|------|------|
| `POST_NOTIFICATIONS` | 发送提醒通知 |
| `VIBRATE` | 震动提醒 |
| `FOREGROUND_SERVICE` | 前台服务保活 |
| `RECEIVE_BOOT_COMPLETED` | 开机自动恢复提醒 |
| `WAKE_LOCK` | 唤醒屏幕显示通知 |
| `SCHEDULE_EXACT_ALARM` | 精确定时闹钟（Android 12+） |
| `REQUEST_IGNORE_BATTERY_OPTIMIZATIONS` | 申请忽略电池优化 |

## 🔋 电池优化设置（重要）

为确保应用在熄屏和后台运行时能正常接收提醒，请务必完成以下设置：

1. **打开手机设置** → **电池** → **后台应用耗电管理**（或类似选项）
2. 找到 **喝水提醒** 应用
3. 将其设置为 **允许后台耗电** 或 **不受电池优化限制**

> ⚠️ 如果不做此设置，应用可能在熄屏后被系统杀死，导致无法接收定时提醒。

**不同品牌手机的设置路径：**

| 品牌 | 设置路径 |
|------|----------|
| 华为 / 荣耀 | 设置 > 电池 > 应用启动管理 > 喝水提醒 > 关闭"自动管理" > 打开"允许后台活动" |
| 小米 / 红米 | 设置 > 电池与性能 > 应用耗电管理 > 喝水提醒 > 设为"无限制" |
| OPPO / 一加 | 设置 > 电池 > 耗电保护 > 喝水提醒 > 关闭"后台冻结"和"深度睡眠" |
| vivo / iQOO | 设置 > 电池 > 后台高耗电 > 喝水提醒 > 开启开关 |
| 三星 | 设置 > 电池和设备维护 > 电池 > 后台应用程序 > 添加喝水提醒 |
| 原生安卓 | 设置 > 电池 > 电池优化 > 不优化 > 选择喝水提醒 |
